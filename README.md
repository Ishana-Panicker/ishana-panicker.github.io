# Ishana Panicker Galaxy Portfolio

A galaxy-themed single-page portfolio website generated from the provided resume.

## Files
- `index.html` — main portfolio page
- `assets/styles.css` — galaxy theme styling
- `assets/script.js` — hover effects
- `assets/Ishana_Panicker_Resume.pdf` — downloadable resume

## GitHub Pages
1. Upload these files to a GitHub repository.
2. Go to Settings → Pages.
3. Select the branch with `index.html` and save.
