# Ishana Premium Galaxy Portfolio

A GitHub Pages-ready static portfolio with a custom purple, blue, and pink galaxy theme.

## Included

- `index.html`
- `assets/css/galaxy.css`
- `assets/css/animations.css`
- `assets/js/stars.js`
- `assets/js/main.js`
- `assets/images/` galaxy SVG artwork
- `assets/icons/` glowing icon SVGs
- `documents/resume.pdf`

## Deploy on GitHub Pages

1. Unzip this folder.
2. Upload all files and folders to your repository root.
3. In GitHub, go to Settings → Pages.
4. Set branch to `main` and folder to `/root`.
5. Save and wait for deployment.

## Customize

Edit `index.html` to change text. Edit `assets/css/galaxy.css` to change colors, layout, and glow effects.
